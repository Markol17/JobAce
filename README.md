# JobTok

JobTok is an app for job seekers to discover jobs and employers to recruit talents combining the TikTok and Tinder experience.

## Getting Started

```cmd
# install dependencies
yarn

# start app
yarn start or expo start

# start Android app
expo start --android

# start iOS app
expo start --ios

# start web version
expo start --web
```
